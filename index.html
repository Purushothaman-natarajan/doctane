Why Doctane?

A unified framework for end-to-end document intelligence with cutting-edge accuracy and speed.

📝

End-to-End OCR

Complete pipeline from text detection to recognition with unified models. Handles straight and rotated text seamlessly.

🔄

Layout Understanding

Automatic page orientation detection and straightening. Multi-column document support with structure preservation.

🏗️

Modular Architecture

Plug-and-play model components. Easy to extend with custom models and preprocessing pipelines.

⚡

High Performance

Optimized inference with PyTorch. DDP training support for multi-GPU acceleration.

🌐

Multi-Language

Support for English, French, and extensible to other languages. Language detection included.

📊

Structured Output

Hierarchical Document objects (Page, Block, Line, Word). Export to JSON and hOCR formats.

Processing Pipeline

Four-stage pipeline for complete document understanding

1

Preprocessing

Page orientation detection and optional straightening for optimal processing.

2

Text Detection

Segmentation-based model identifies word/line bounding boxes (straight & rotated).

3

Text Recognition

Crops fed to recognition model for accurate text transcription.

4

Document Assembly

Structured output with geometry, confidence scores, and KIE support.
