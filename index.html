Processing Pipeline

Four-stage pipeline for complete document understanding

1

Preprocessing

Page orientation detection and optional straightening.

2

Text Detection

Segmentation-based model identifies text regions.

3

Text Recognition

Crops fed to recognition model for transcription.

4

Document Assembly

Structured output with geometry and confidence scores.

Quick Start

Get up and running in minutes

# Clone the repository git clone https://github.com/Purushothaman-natarajan/doctane.git cd doctane # Install dependencies pip install -r requirements.txt # Start the API server python api/main.py
1

Clone

Download the repository from GitHub

2

Install

Run pip install to get all dependencies

3

Launch

Start API server and open http://localhost:8000/app

Important Notes

🔧 No Pre-trained Weights

We provide the code only, not pre-trained model weights. You need to train your own models using the provided training scripts.

💻 Bring Your Infrastructure

Training requires GPU resources. You must have your own infrastructure (local GPU, cloud like AWS/GCP/Azure). We support DDP for multi-GPU.
