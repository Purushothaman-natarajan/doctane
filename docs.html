Documentation

Comprehensive guide to Doctane's features, API, and usage.

Installation

Install Doctane from source or via pip:

# From source git clone https://github.com/Purushothaman-natarajan/doctane.git cd doctane pip install -e . # Or install from PyPI pip install doctane

Quick Start

Run OCR on an image with just a few lines of code:

import numpy as np from PIL import Image from doctane.ocr_pipeline.ocr_predictor import OCRPredictor # Load image image = Image.open("document.jpg").convert("RGB") np_image = np.array(image) # Run OCR predictor = OCRPredictor() output = predictor([np_image]) # Access results for page in output.pages: print(page)

Configuration

Doctane uses YAML configuration files for model and training settings.

detection: model: seg_linknet_resnet50 input_size: [512, 512] batch_size: 8 recognition: model: sar_resnet34 vocab_size: 95 batch_size: 32 training: epochs: 100 lr: 0.001 optimizer: adam

Detection Models

Available segmentation-based detection models:

  • LinkNet - Fast and lightweight
  • DeepLabV3+ - High accuracy for complex layouts
  • SegFormer - State-of-the-art transformer-based

Recognition Models

Text recognition models for transcription:

  • SAR - Sequence Approximation Recognition
  • ViTSTR - Vision Transformer for STR
  • CRNN - CNN + RNN + CTC
  • MASTER - Multi-Aspect Self-Attention

Training

Train detection model:

python train/text_detection/train_detection.py \ --config configs/detection.yaml \ --epochs 100

Train recognition model:

python train/text_recognition/train_recognition.py \ --config configs/recognition.yaml \ --epochs 50

Running Inference

Use the provided inference script:

python scripts/run_inference.py \ --input_dir data/raw \ --output_dir results/outputs
