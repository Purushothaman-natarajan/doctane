Developer Guide

Complete guide to setting up development environment and contributing to Doctane

🔧 Development Setup

1 Clone Repository

git clone https://github.com/Purushothaman-natarajan/doctane.git cd doctane

2 Create Virtual Environment

python -m venv venv venv\Scripts\activate # Windows # source venv/bin/activate # Linux/Mac

3 Install Dependencies

pip install -r requirements.txt pip install -e .

4 Run Development Server

python api/main.py

⚙️ Environment Variables

VariableDescriptionDefault
BUILD_VERSIONPackage version string0.1.0a0
CHECKPOINT_DIRModel checkpoints directory./checkpoints
DATA_DIRTraining data directory./data
LOG_DIRTraining logs directory./logs

🧪 Running Tests

# Run all tests pytest tests/ # Run specific test file pytest tests/test_detection.py -v # Run with coverage pytest tests/ --cov=doctane --cov-report=html

🏋️ Training Models

Text Detection Training

python train/text_detection/train_detection.py \ --config configs/detection.yaml \ --epochs 100 \ --batch_size 8 \ --lr 0.001

Text Recognition Training

python train/text_recognition/train_recognition.py \ --config configs/recognition.yaml \ --epochs 50 \ --batch_size 32

Distributed Training (DDP)

torchrun --nproc_per_node=4 \ train/text_detection/train_detection_ddp.py \ --config configs/detection.yaml

🚀 Inference

Command Line

python scripts/run_inference.py \ --input_dir data/raw \ --output_dir results/outputs \ --model_type seg_linknet_resnet50

Python API

from doctane.ocr_pipeline.ocr_predictor import OCRPredictor predictor = OCRPredictor() results = predictor([image_array]) results.show()
